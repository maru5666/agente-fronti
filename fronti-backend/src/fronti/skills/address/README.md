# Skill: Dirección

Detecta direcciones escritas por el cliente y evita que Fronti las trate como productos.

Activa con frases como:

- "Estoy en Barrio Obrero"
- "Mi dirección es Calle 16"
- "Barrio Sucre"

Salida esperada:

- Confirma recepción de la dirección.
- Marca que debe verificarse con Maps o delivery.
- Mantiene separación por `companyId` desde el contexto del agente.
