# Skill: Facturación

Genera comprobantes PDF básicos con fecha y hora.

Responsabilidades:

- Buscar el pedido reciente del cliente.
- Generar PDF en `storage/invoices/{companyId}`.
- Registrar ruta generada en logs de Skill.

Nota: esta Skill está preparada para evolucionar a facturación fiscal o integración con proveedor externo.
