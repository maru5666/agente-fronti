# Skill: OCR BCV

Consulta la tasa oficial del Banco Central de Venezuela.

Responsabilidades:

- Abrir la fuente oficial configurada.
- Intentar leer imagen oficial por OCR.
- Nunca usar EUR ni otras monedas.
- Guardar historial con tasa, fuente, fecha, hora, imagen y método de extracción.

Métodos posibles:

- `ocr_official_image`
- `html_official_page`
